Welcome to Chainrush!

HOW TO USE:
1) Install python onto your machine.
2) Extract the contents of the .zip file into one directory.
3) Right-click main.py and click run to play!

IMPORTANT NOTES:
- The only manager available in this version is Chainsaw Consultant (hence the name Chainrush).
- You can edit the values in ManagerGlobals.py to your liking.
